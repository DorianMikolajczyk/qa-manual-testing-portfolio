# Test Plan – E-commerce Web Application

## 1. Object (Co testujemy)

Testowanie funkcjonalności sklepu internetowego.

## 2. Scope (Zakres testów)

W ramach testów sprawdzane będą:

- rejestracja użytkownika
- logowanie użytkownika
- wyszukiwanie produktów
- dodawanie produktów do koszyka
- usuwanie produktów z koszyka
- proces składania zamówienia

## 3. Test Types

- Testy funkcjonalne
- Testy pozytywne
- Testy negatywne
- Testy regresyjne
- Testy smoke

## 4. Assumptions (Założenia)

- Aplikacja jest dostępna dla użytkownika
- Użytkownik posiada stabilne połączenie internetowe
- Dane testowe są dostępne
- Środowisko testowe działa poprawnie

## 5. Out of Scope (Poza zakresem)

- Testowanie panelu administratora
- Testowanie kodu aplikacji
- Testowanie infrastruktury serwerowej
